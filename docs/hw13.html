Question 1

I used the Burnham_field_data to look at if different sampling events have an impact on disease abundance.

This was a non-gaussian (binomial function) and used an interaction term (do sampling events affect disease abundance)

library(dplyr)

bee_dat_q1 <- bee_dat %>%
  select(sampling_event,
    BQCV_pathogen_binary,
    DWV_pathogen_binary,
    Nosema_pathogen_binary) %>%
  pivot_longer(
    cols = c(BQCV_pathogen_binary, DWV_pathogen_binary, Nosema_pathogen_binary),
    names_to = "disease",
    values_to = "abundance"
  )

q1_plot <- ggplot(bee_dat_q1, aes(x = sampling_event, y = abundance, color = disease))+
  geom_smooth(method = "glm", method.args = list(family = binomial))+

    title = "Disease Abundance Across Different Sampling Events",
    y = "Disease Abundance",
    x = "Sampling Event",
    color = "Disease"

Sampling events were catagorical, and when sampling you either have or don’t have the disease, so I used this plot to show a binary relationship if a disease was present during the sampling event

The p-value was 0.02134, which is not statistially significant, so sampling events don’t affect disease abundance. I expected the data to not be statistically significant, as I didn’t think the sampling events occured over a long period of time.

However, if the sampling event occured over a longer time period, then it may have been interesting to see if disease abundance was changing over time, but I don’t know the time frame of sampling events
